# Potential-Enigma

## JS Project

```Shell Session
# install dependencies
yarn 

# development server
yarn start

# build for production
yarn build
```

### Using Eslint & Prettier

With VSCode, put this config in `.vscode/settings.json`

```json
{
    "eslint.validate":[
      {
        "language":"javascript",
        "autoFix":true
      }
    ],
    "eslint.autoFixOnSave":true,
    "editor.formatOnSave":false
  }
```
